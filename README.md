# Twitter / X Search Builder - Runnable React Project

## איך מריצים

1. חלץ את קובץ ה-ZIP.
2. פתח טרמינל בתיקיית הפרויקט.
3. הרץ:

```bash
npm install
npm run dev
```

4. פתח בדפדפן את הכתובת ש-Vite מציג, בדרך כלל:

```text
http://localhost:5173
```

## מה תוקן ביחס לקוד שהודבק בצ'אט

- הוסר HTML שנכנס בטעות לתוך JSX סביב קישורים.
- תוקנו קריאות `window.open`.
- תוקנה יצירת URL לחיפוש ב-X/Twitter.
- הופרד הפרויקט למבנה תקין של Vite: `index.html`, `src/main.jsx`, `src/App.jsx`.
- נוספו `package.json` וסקריפטים להרצה.

## הערה

הפרויקט משתמש ב-React, Vite ו-lucide-react.
